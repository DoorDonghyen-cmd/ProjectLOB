## 📋 상태 정의
*   **Open (대기)**: 현상이 제보되었고, 재현이 확인되어 해결 예정인 상태.
*   **Fixed (진행)**: 코드 혹은 설정 수정이 완료되어 컴파일 완료 상태이나 아직 테스트 확인 전.
*   **Verified (검증)**: 수정 후 정상 동작이 직접 검증되었으며 유저 확인을 대기 중인 상태.
*   **Closed (해결)**: 검증 완료 후 정상 동작이 확인되어 완전히 종결된 상태.
