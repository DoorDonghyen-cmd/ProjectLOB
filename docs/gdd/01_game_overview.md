## 1. 게임 개요

### 1.1 엘리베이터 피치

> **봉쇄된 빌딩에 침투한 특수 작전 요원이 진입 전 전술 탄환의 장전 순서를 정해, 좁은 복도를 따라 조여오는 무장 적들을 제압하며 최상층의 탈출 헬기(Boarding)를 향해 나아가는 순수 스탯 전술 덱빌더.**
> 피지컬 반사신경이 아닌, *장전 순서 설계*와 *스탯 정산*의 전술로 생존한다.

### 1.2 기본 정보

| 항목 | 내용 |
|---|---|
| **장르** | 로그라이크 런 기반 · 순수 스탯 전술 덱빌더 (실시간 액션 없음) |
| **플랫폼** | 모바일 (iOS/Android) · PC |
| **엔진** | Godot 4.x |
| **타겟 스코프** | 1인/소규모 상용 사이드 프로젝트 |
| **핵심 판타지** | "방에 돌입하기 전 탄창을 완벽히 셋업하여 복도의 위협을 제압하는 특수요원" |

### 1.3 세계관 및 상황 설정

- **배경**: 불명예스러운 생화학 무기/나노 테러 또는 미승인 AI 실험으로 인해 완전히 봉쇄(Lockdown)된 다국적 기업의 초고층 본사 빌딩.
- **주인공**: 빌딩 내 핵심 데이터를 회수하고 사태를 종식하기 위해 단독 투입된 특수 작전 요원 (Agent).
- **제목 "Last on Board"의 재해석**:
  - 빌딩 옥상의 **탈출 헬기(Boarding)**에 무사히 탑승해 살아남아야 하는 *마지막 요원*.
  - 또는 작전 통제실의 **작전 보드(Tactical Board)** 위에 살아남은 *마지막 생존 요원*.

### 1.4 디자인 필라 (Design Pillars)

1. **장전 순서가 곧 사격 작전이다** — 방에 진입(Breaching)하기 전, 적들의 예상 배치와 스탯에 맞춰 탄창의 급탄 순서를 설계해야 한다.
2. **전술적 계산은 투명하다** — 플레이어가 피해량 및 명중을 완전하게 예측하고 통제할 수 있다.
3. **복도 거리가 곧 생명선이다** — 좁은 사무실 복도나 환기구에서 적과 대치하며, 적과 닿는 것(거리 0)은 제압(Takedown) 및 즉사를 의미한다. 넉백탄과 비살상 경직탄이 거리를 지키는 핵심 방어 수단이다.
4. **재장전(Reload)은 위험 부담이다** — 엄폐물 없이 탄창을 교체(Tactical Reload)해야 하는 긴박한 턴 소모는 적에게 거리를 내주는 기회비용이다.

### 1.5 전체 플레이 루프 (Core Game Loop & Flow)

본 게임의 진행은 크게 3가지 중첩된 루프(메타, 런, 전투)로 순환하며, 각 단계는 디제틱한 특수요원의 작전 단계와 유기적으로 호환된다.

#### ① 3단계 게임 루프 구조

```
[1. 메타 루프: 기지 & 영구 성장]
  │   (작전 투입)
  ▼
[2. 런 루프: 빌딩 침투 & 노드 이동] ◀────────────────────────┐
  │   (방 진입 / Breaching)                                │
  ▼                                                        │
[3. 전투 루프: 1차원 복도 대치] ◀───────────┐                │
  │   (발사/Unload 정산)                    │ (전투 루프 순환)│ (노드 완료)
  └─────────────────────────────────────────┘                │
  │   (전투 승리 보상: 탄환 드래프트)                         │
  └──────────────────────────────────────────────────────────┘
  │   (옥상 헬기 탑승 클리어 or 사망)
  ▼
[작전 보고 및 메타 정산 (Debriefing)]
  │   (언락 및 영구 강화)
  └──────────────────────────────────────────────────────────┘
```

#### ② 상세 페이즈 흐름 (Phase Flow)

| 단계 | 페이즈명 | 주요 조작 및 플레이어 경험 | 디제틱 연출 및 의미 |
|---|---|---|---|
| **메타 단계** | **1. 메인 화면 및 요원 관리** | 요원의 영구 업그레이드 현황 점검, 새로운 전술 탄환 언락, 기본 권총 및 기초 탄환 세트 준비. | 작전 통제실에서의 브리핑 및 준비 단계 |
| **런 단계** | **2. 빌딩 침투 (침투 개시)** | 초고층 빌딩 단면도 맵에서 침투 경로 선택. 위로(또는 아래로) 향하는 비상계단, 환기구, 엘리베이터 샤프트 분기 노출. | 빌딩 외벽/지하 침투 및 내부 진입 |
| | **3. 전술 정비 (안전 구역)** | 무기 캐비닛 노드 또는 엄폐 휴식 구역 상호작용. 보유 탄고(Deck) 카드 강화(장약 개조) 또는 복제/삭제. | 빌딩 내부 안전실에서 장비 점검 |
| **전투 단계** | **4. 장전 페이즈 (Loading)** | 전투 인카운터(사무실/서버실 등) 진입 전, 문 너머 적의 정보(아키타입, 체력, 태세 전환 조건) 확인 후 **LIFO(스택) 역순 삽탄 장전**. | 방 문 뒤에 밀착하여 침투 계획을 세우는 CQB 전술 행동 |
| | **5. 교전 페이즈 (Engage)** | 1차원 트랙 대치 상황에서 **격발(Shoot)** 또는 **Unload(비용 지불: 탄 소실 + 적 즉시 전진)** 선택 및 라이브 프로젝션 확인. | 좁은 복도 코너에서의 물리적 교전 및 사격 |
| | **6. 정산 페이즈 (Result)** | 적 무력화 성공 시 전술 탄환 드래프트 보상 획득 및 다음 층 단면도 맵으로 복귀. | 해당 층 보안 구역 제어권 확보 |
| **정산 단계** | **7. 작전 디브리핑 (런 종료)** | 요원의 사망 혹은 최상층 옥상의 보스를 제압하고 **탈출 헬기(Boarding)**에 무사히 탑승 시 작전 결과 보고서 연출 및 데이터 칩 획득. | 헬기를 타고 탈출하거나 전사하여 작전이 기록됨 |

## 부록: 용어 사전

| 용어 | 정의 |
|---|---|
| 총알 (Bullet) | 드래프트하는 카드. DMG/ACC/PEN 스탯 보유 |
| 탄창 (Magazine) | 순서 있는 총알 컨테이너. 행동 예산 |
| 총 (Gun) | 규칙 엔진. 탄창 크기·구조·리로드 방식·패시브 결정 |
| 리로드 (Reload) | 탄창 재충전. 턴 소모 = 적 전진 비용 |
| 거리 트랙 (Distance Track) | 1차원 이산 정수 전장 |
| 런 (Run) | 한 번의 로그라이크 플레이스루 |
| 전투씬 (CombatScene) | 런 도중의 개별 전투(인카운터)를 처리하는 통합 씬 |
| 장전 페이즈 (Loading) | 전투씬 진입 직후, 적 정보 확인 및 LIFO 역순 삽탄 장전 단계 |
| 교전 페이즈 (Engage) | 발사/빼내기/리로드 턴 루프가 진행되는 실전 교전 단계 |
| 정산 페이즈 (Result) | 전투 승리 후 보상 드래프트 및 다음 노드 복귀 단계 |
| 넉백 (Knockback) | 적을 N칸 뒤로 밀어 거리 회복 |
| 둔화 (Slow) | 적의 다음 턴 이동속도 감소 |
| 드래프트 (Draft) | 런 도중 총알을 선택·획득하는 행위 |
