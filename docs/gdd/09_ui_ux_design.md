## 9. UI/UX 설계

본 게임은 **가로형 횡스크롤 측면 뷰(Landscape Side-View)**를 표준 화면 방향으로 채택합니다. 5~6등신의 리얼 비례 픽셀 캐릭터들의 사격 및 재장전 애니메이션 쾌감을 극대화하고, 요원과 좀비들 간의 직선 거리 트랙을 한눈에 투명하게 인지할 수 있도록 설계되었습니다.

---

### 9.1 가로형 기본 레이아웃 구성 (Landscape Layout Grid)

전체 화면은 크게 **좌측 전술 패널(35%)**과 **우측 전장/인벤토리 패널(65%)**의 HBox 구조로 나뉩니다.

┌────────────────────────────────────────────────────────┐
│  [ 5층: 생화학 연구실 복도 ]                             │
│                                                        │
│  [요원 요성] ─── (3m) ─── [좀비 A] ─────── (6m) ─── [술사] │
│  HP: 20/20                  HP: 12/12                  HP: 8/8 │
│  [탄창 튜브]                (이동력: 1)                (차징: 2T)│
│  ┌──────┐                                              │
│  │ AP   │                                              │
│  ├──────┤                                              │
│  │ JHP  │                                              │

│  │ JHP  │       [ 가방 카드 인벤토리 ]                 │
│  └──────┘    ┌────────┐  ┌────────┐  ┌────────┐        │

### 9.2 인게임 전투씬 구성 요소

#### ① 우측 상단: 가로형 횡스크롤 복도 영역 (Horizontal Corridor View)
- **카메라 및 캐릭터 시점**: 사이드 뷰(Side-Profile View) 횡스크롤 복도. 요원은 항상 복도 좌측에 엄폐하거나 서서 사격 자세를 취하며, 적 좀비들은 복도 우측에서 생성되어 좌측(요원 쪽)으로 접근합니다.
- **공유 거리 트랙**: 요원과의 남은 거리(m)가 캐릭터 머리 위에 수치 및 보드판 상의 눈금 간격으로 직관적으로 표시됩니다.
- **예측 고스트 토큰**: 턴 종료 시 혹은 발사 예약 시, 적들이 다음 턴에 이동할 예정 위치(고스트 토큰)와 예상 피격 HP 게이지를 실시간으로 투영해 보여줍니다.

#### ② 좌측 하단: 탄창 실린더 및 기본 액션 패널 (Magazine HUD)
- **LIFO 탄창 튜브**: 투명한 튜브 형태로 표시되어 있으며, 삽탄된 탄들이 아래에서부터 겹쳐서 적재되고 **발사 시 가장 위(Top)에 있는 탄환이 튀어나가며 소모**되는 연출을 제공합니다.
- **적재 비주얼**: 각 탄환 아이콘(`JHP`, `AP`, `SLG` 등)은 구경별/패시브별 고유 픽셀 그래픽으로 나타나며, 음수 간격(`separation = -12`)을 통해 탄창 내에 촘촘하게 맞물려 들어간 입체감을 묘사합니다.
- **핵심 액션 버튼**:
  - `[발사 (Fire)]`: 맨 위의 탄을 격발합니다.
  - `[비움 (Eject/Unload)]`: 약실 내 탄을 버립니다. (탄 소실 및 적 즉시 1칸 접근 패널티 발생)
  - `[재장전 (Reload)]`: 덱에서 탄을 일괄 공급받아 재정비합니다.

#### ③ 우측 하단: 가방 카드 인벤토리 (Inventory & Deck Panel)
- **장전 페이즈 (Loading Phase)**: 가방 패널이 위로 확장(`height = 240px`)되며 카드 덱이 활성화됩니다. 플레이어는 카드를 드래그하여 좌측 탄창 튜브에 드롭하여 삽탄을 설계합니다.
- **교전 페이즈 (Engage Phase)**: 가방 패널이 아래로 접히고(`height = 80px`), 현재 덱의 남은 매수 요약 정보만 간결하게 표시하여 인게임 전장의 복도 횡스크롤 뷰를 최대한 넓고 웅장하게 보여줍니다.

---

### 9.3 드래그 앤 드롭 및 조작 UX (Loading UX)

- **드래그 앤 드롭 삽탄**: 가방에서 탄환 카드를 누른 채로 좌측 탄창 튜브로 끌어다 놓으면 튜브 맨 위에 총알이 추가 적재됩니다.
- **장전 확인 및 확인 팝업**: 탄창의 삽탄 한계를 넘길 수 없으며, 모든 설계를 마친 후 `[전투 시작]` 버튼을 누르면 교전 페이즈로 전환됩니다.
- **직관적 탭 취소(Undo)**: `[비우기]` 버튼 또는 튜브 맨 위 총알을 탭하면 가장 최근에 장전한 총알부터 순차적으로 꺼내 가방으로 돌려놓을 수 있습니다. (장전 페이즈 전용)

---

### 9.4 전투 연출 및 템포 (Combat FX)

1.  **발사(Fire) 이펙트**: 격발 시 요원의 총구에서 화염 플래시(Muzzle Flash)가 번쩍이고 화면에 미세한 진동(Screen Shake)이 가해집니다.
2.  **LIFO 탄환 소모**: 격발 즉시 탄창 내 최상단 탄환 이미지가 살짝 위로 튕겼다가 사라지며, 아래에 대기 중이던 탄환들이 튀어오르는(Bouncing Tween) 연출과 함께 순차적으로 올라옵니다.
3.  **적 피격 및 넉백**: 탄환 속성에 따라 탄이 날아가 피격되는 궤적이 그려지고, 넉백 탄 격발 시 좀비 스프라이트가 부드러운 트윈 모션과 함께 뒤로 미끄러지는 시각 효과를 묘사합니다.
